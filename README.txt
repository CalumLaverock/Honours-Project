Controls:
Escape - Close the program
Space - Automatically run all steps of program until a suitable map is produced
D - Activate debug mode

Debug Mode Controls:
G - Generate rooms
S - Separate rooms
R - Select all rooms above 75 units on the x and y axis, and with 3 or more rooms touching it
O - Determine the objective rooms from previously selected rooms
A - Select all rooms above 75 units on the x and y axis, and with 2 or more rooms touching it
P - Determine the spawn rooms from previously selected rooms
C - Connect the important rooms to one another and draw the bounding rectangles to fill in gaps between rooms

N - Only draw the selected rooms
B - Highlight the connecting bounding rectangles
